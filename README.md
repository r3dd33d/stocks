# ML Project – Stock Trading Signals

## Dataset
Download `kairognos_ml_dataset_with_mktcap.parquet` and place it in the `data/` folder.

## Project Structure
```
├── data/                  # Data files (not tracked by git)
├── notebooks/
│   └── 01_data_loading.ipynb
├── requirements.txt
└── README.md
```

## Setup
```bash
pip install -r requirements.txt
```
